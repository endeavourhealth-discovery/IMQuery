## Run Frontend (development)

Checkout from GIT

`cd frontend-vue`

`npm install` (you only have to do this once)

`npm run serve`

Navigate your browser to `http://localhost:8081`

The proxy expects the API to be running on localhost:8080

## Build Frontend (production)

`npm run build`
